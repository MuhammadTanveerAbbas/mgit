# Contributing to mgit

Thanks for your interest. This is a tool built and maintained by
[The MVP Guy](https://themvpguy.vercel.app) — Muhammad Tanveer Abbas.

## Reporting Bugs
Open an issue with your OS, Node version, and the exact command you ran.

## Pull Requests
Keep PRs small and focused. One fix per PR.
Run `node bin/mgit.js` to verify your change works before submitting.

## Code Style
- ESM modules only (no CommonJS)
- No TypeScript
- Use the existing `ui.js` helpers for all terminal output — no raw console.log
